HTML注释：

  • 可以将注释插入 HTML 代码中，这样可以提高其可读性，使代码更易被人理解。浏览器会忽略注释，也不会显示它们。
  • 开始括号之后（左边的括号）需要紧跟一个叹号，结束括号之前（右边的括号）不需要。
  • 语法：<!--注释内容-->
  • 合理地使用注释可以对未来的代码编辑工作产生帮助。

锚标记：

  • 相当于书签，当使用锚标记时，我们可以创建直接跳至该命名锚的链接
  • 语法：< a name=“自定义名字">锚（显示在页面上的文本）</a>
  • 创建使用该锚的链接：< a href=“#自定义的锚名字”>显示内容

图片标签：

  • < img />标签将其他文件夹或服务器的图片显示到网页中
  • < img />标签只包含属性，并且没有闭合标签结尾加反斜杠
  • 语法：< img src=“图片地址”title="可见提示文本" width="图片宽度" height="图片高度" alt=“自定义替代文本"/>
  • src:属性指定图片文件的路径，分为绝对路径和相对路径。相对路径
  • 绝对路径：指带域名的文件的完整路径。
  • 相对路径：同一个目录的文件引用如果源文件和引用文件在同一个目录里，直接写引用文件名即可。
  • width和height来指定图片尺寸 注意：在网页中插入图片必须加上width宽度height高度否则网页在浏览器中打开时就会重新计算图片大小如果网页中有张图片这样就会严重影响网页打开的速度。
  • title属性，指鼠标悬停时的可见提示文本，主要用于增加用户体验，并且只能提示文本：
  • alt属性:表示图片无法显示时的说明性替代文字 如下：
  • < img src="images/lsdkglskgl.jpg" alt="图片提示文本" width="270" height="129"/>
