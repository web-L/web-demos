select 下拉列表

下拉列表：定义一个框架集

语法： 
		< select   name="" size="3" disabled="">

			< option value=“apple”>苹果</option> 	

			< option value=“pear”>梨子</option>

			< option value=“banana”>香蕉</option> 

			< option value=“peach”>桃子</option>

		 </select>

name="下拉列表名称"
size="下拉列表可视的选项数目"
disabled="禁用读下拉列表"
